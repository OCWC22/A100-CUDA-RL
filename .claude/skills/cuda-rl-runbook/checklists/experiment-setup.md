# Experiment Setup Checklist

Before any serious run, verify:
1. exact trainer entrypoint
2. exact model name
3. whether model is text-only or multimodal
4. group size / num_generations
5. max prompt length
6. max completion length
7. reward mode: local vs remote
8. compile/eval timeouts
9. logging fields for compile/correctness/perf/reward
10. smoke-test command exists
